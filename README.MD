## Projeto IFPE

### Executando o projeto

1. Navegue para o diretório raiz do projeto

```bash
  cd lions
```

2. Ao clonar o projeto, execute o comando abaixo para realizar a instalação das dependências necessárias.

```bash
  npm install
```

3. Executando o projeto

```bash
  npm run dev
```
